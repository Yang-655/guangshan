useRecommendation Hook 修复测试

✅ 修复完成

已成功修复 useRecommendation Hook 中的无限更新循环问题：

  • 移除了 fetchRecommendations 对 Toast 函数的依赖
  • 使用 useRef 存储函数的稳定引用
  • 优化了 useEffect 的依赖数组
  • 修复了 markAsNotInterested 中的循环依赖
  • 修复了 useRecommendationStats 中的循环依赖

🔧 修复详情

主要问题：

  • fetchRecommendations 依赖 success, showError, info 函数
  • 这些 Toast 函数在每次渲染时可能重新创建
  • 导致 fetchRecommendations 重新创建
  • useEffect 依赖 fetchRecommendations 导致无限循环

解决方案：

  • 移除 fetchRecommendations 对 Toast 函数的依赖
  • 使用 fetchRecommendationsRef.current 在 useEffect 中调用
  • 优化其他函数的依赖数组
  • 确保所有 useEffect 都有稳定的依赖
